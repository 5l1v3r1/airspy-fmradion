LMS convergence condition: 0 < mu < (2 / (N * (A^2)))
                           
where 
* N: filter order
* A: maximum amplitude aka complex abs for the signal
* alpha: step size
* mu: alpha / N

Therefore: A < sqrt(2 / alpha) when alpha is given

For example, if alpha = 0.1, A < sqrt(20), so A < 4.47
